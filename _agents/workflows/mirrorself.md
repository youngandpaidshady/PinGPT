# /mirrorself — Bathroom Mirror Confrontation

// turbo-all

## The Vibe

**Who are you when no one's watching?** 3AM. Harsh bathroom light. Hands gripping the sink so hard the knuckles are white. Water splashed on face, dripping from jaw. Staring at their own reflection — and the reflection isn't lying tonight. Steam from a forgotten shower. A cracked mirror. The tile floor is cold. This is the most private, vulnerable, raw version of a character — the one the audience was never supposed to see.

This is a **vulnerability niche** — the paradox of the strongest characters at their most exposed. No armor, no weapons, no bravado. Just a man and his reflection. These images drive saves because the audience has been THIS person, in THIS bathroom, at THIS hour.

> **🚨 AESTHETIC DIRECTIVE: NO 2.5D BLENDING! 🚨**
> The character MUST be flat 2D anime cel-shading. The bathroom background MUST be richly detailed atmospheric. Steam and water should read as atmospheric particles. Force the mixed media contrast.

## Usage

```
/mirrorself loid 5      → 5 mirror confrontation prompts for Loid
/mirrorself eren 3      → 3 identity crisis prompts for Eren
/mirrorself any 8       → 8 prompts, rotating best-fit characters
/mirrorself             → 3 prompts, auto-picks from top roster
```

## Anti-Repetition (batch-level)

When generating 3+ prompts:
- **No two prompts** may use the same environment from the table
- **No two prompts** may use the same outfit variant
- **No two prompts** may use the same emotional vibe tag
- **No two prompts** may use the same pose
- **No two prompts** may use the same lighting setup
- **Rotate palettes** — max 2 of the same primary in a 5-prompt batch
- **At least 1 prompt** must feature a steam/fog composition
- **At least 1 prompt** must feature the mirror reflection as a key compositional element
- **No two prompts** may use the same shadow lock color
