# /skylineache — Twilight Elevation

// turbo-all

## The Vibe

**The world is huge. He's standing in it.** Rooftop railing at sunset, leaning forward, city sprawling below like a circuit board someone forgot to turn off. Open field at twilight, headphones on, head tilted back, one hand reaching for a purple sky that doesn't reach back. Crescent moon over a sleeping city. This isn't indoor loneliness — this is **outdoor loneliness**, the kind where the vastness of the world makes the solitude hit different. He's not hiding in a room. He's standing in the open and still somehow invisible.

This is the **elevated contemplation niche** — the character dwarfed by landscape, composition emphasizing the gap between one person and the entire world. The purple-pink twilight palette is *aggressively* Pinterest-coded: it converts at 2x the rate of standard blue/grey moods because it photographs as beautiful before it registers as sad. The audience saves for the COLOR first, then reads the emotion. Double hook.

> **🚨 AESTHETIC DIRECTIVE: NO 2.5D BLENDING! 🚨**
> Character = flat 2D anime cel-shading. Sky/landscape/cityscape = richly detailed atmospheric. Twilight gradient = atmospheric. Force the mixed media contrast.

## Usage

```
/skylineache megumi 5     → 5 skyline prompts for Megumi
/skylineache any 8        → 8 prompts rotating characters
/skylineache              → 3 prompts, auto-picks from top roster
```
